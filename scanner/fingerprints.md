# 指纹库（fingerprints）

## 后端
- Maven：pom.xml
- Gradle：build.gradle / build.gradle.kts
- MyBatis：*Mapper.xml / @Mapper / mybatis-config.xml / 依赖含 mybatis
- MyBatis-Plus：依赖含 mybatis-plus
- JPA：spring-data-jpa
- Flyway：db/migration 目录
- Liquibase：db/changelog 目录、liquibase.properties

## 前端
- pnpm：pnpm-lock.yaml
- yarn：yarn.lock
- npm：package-lock.json
- TypeScript：tsconfig.json
- Vite：vite.config.*
- Vue3：vue / @vue/*
- ESLint：eslint.config.* / .eslintrc.*
- Prettier：.prettierrc*
